Autonomous IoT Car Challenge using Computer Vision and ESP32
Overview

This project explores the development of an autonomous vehicle prototype using Computer Vision, Python, and ESP32 IoT technologies. The main objective was to create a low-cost intelligent robotic car capable of interacting with its environment, processing visual information, and executing real-time commands.

The project combines concepts from:

Artificial Intelligence
Embedded Systems
Internet of Things (IoT)
Computer Vision
Robotics
Real-Time Communication

This repository was developed as a practical and research-oriented project, suitable for students interested in pursuing a Master’s Degree in areas such as:

Computer Science
Robotics
Artificial Intelligence
Embedded Systems
Intelligent Transportation Systems
IoT Research
